# Pagination

Pagination for Padrino framework:

* brutal
* classic
* digg
* extended
* punbb

## Installation

Add this line to your application's Gemfile:

    gem 'padrino-pagination'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install padrino-pagination
